# EcoPulse Smart Energy Management System

An eco-friendly hybrid renewable energy management project for monitoring and controlling energy-related data across the system.

## Project Structure

- `arduino/` - Arduino controller code
- `backend/` - server and API files
- `frontend/` - Python and Streamlit app files
- `EcoPulse-Streamlit-Flask/` - project assets and related app files

## Notes

This repository contains multiple parts of the EcoPulse project. Open the folder that matches the part you want to run or edit.
